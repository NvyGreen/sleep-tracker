# Sleep Tracker
This project allows a user to track the amount of sleep they get a night, as well as their sleepiness levels in the daytime.

## Tech Stack
Frontend: HTML/CSS, JavaScript, Ionic

## Key Features
Through this app, users can
- Log the start and end of their sleeping time
- Log how sleepy they feel throughout the day using the Stanford Sleepiness Scale

## Running Locally
To run this project locally:
1. Clone the repository and navigate to the project directory
2. In the terminal, install the Ionic CLI using `npm install -g @ionic/cli`
3. Install the package dependencies using `npm install`
4. Run `ionic serve`
